## How to Upgrade

Follow these steps to ensure you have the latest version of the Vandar client or integration.

<Steps>
  <Step title="Check Current Version" icon="package">
    Run the version check command in your terminal.

````bash
vandar --version
````

  </Step>
  <Step title="Update Package" icon="download">
    <Tabs>
      <Tab title="npm" icon="nodejs">
````bash
npm update vandar-store-sdk
````
      </Tab>
      <Tab title="yarn" icon="package">
````bash
yarn upgrade vandar-store-sdk
````
      </Tab>
    </Tabs>
  </Step>
  <Step title="Verify Installation" icon="check-circle">
    Test a sample purchase flow using the dashboard at `https://dashboard.vandarstore.com`.
  </Step>
</Steps>
